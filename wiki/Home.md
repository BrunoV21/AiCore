## Overview

The AiCore project offers:
- A standardized API to work with various LLM providers (OpenAI, Mistral, Groq, Gemini, Nvidia, OpenRouter) and embedding solutions.
- Integrated logging and streaming of LLM outputs.
- A comprehensive observability module to track operations, monitor token usage, and analyze costs.
- Support for reasoning augmentation to improve the quality of AI responses.
