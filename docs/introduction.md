
# Introduction

Welcome to the AiCore Project Documentation. This documentation serves as a comprehensive guide for setting up and using AiCore to build and deploy AI applications. The AiCore Project provides a unified and consistent API to interact with multiple language model (LLM) and embedding providers.

## Project Overview

AiCore is designed to offer:
- A consistent API for both synchronous and asynchronous operations.
- Integration with various LLM and embedding providers for streamlined AI workflows.
- Advanced reasoning augmentation to enhance the processing of complex tasks.
- Built-in observability and logging to monitor performance, usage, and resource management.
- Scalability for deployment in high-throughput, production environments.

By leveraging modern Python best practices and a modular architecture, AiCore ensures maintainability, extensibility, and reliable operation across diverse AI applications.

## Key Features

- Unified interface for multiple AI providers
- Support for synchronous and asynchronous operations
- Advanced reasoning and configuration options
- Integrated observability with detailed metrics and an interactive dashboard
- Easy extensibility for incorporating future enhancements and additional modules
